How this works

  1. Enter trading days, current NIFTY, previous close, and current India VIX.
  2. The site computes the current daily log return from current NIFTY and previous close.
  3. Each model runs Monte Carlo simulation using the latest monthly-updated JSON model objects.
  4. The output shows the 15th, 50th, and 85th percentile forecast range and band width percentages.
